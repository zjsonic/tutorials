# Value

## 值
- 值就是数据
- 值就是信息
- 计算机程序的运行，离不开对“值”进行操作。


## 值是有类型的
值是有类型的，能够表示并操作的值的类型称作数据类型。
编程语言的最基本的特征就是能够支持多种值。
- pimitive type
  - number
  - string
  - boolean
  - undefined
  - null
  - symbol
- object type
  - object：“名值对”的无序集合
  - array: “名值对”的有序集合
  - function: 具有可执行代码的对象
  - 包装对象
