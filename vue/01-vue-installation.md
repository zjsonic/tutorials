# Vue Installation

## 直接下载引入
```
<script src = './js/vue.js'></script>
```
- `Vue`会被注册为一个全局变量。

## 使用CDN引入
```
<script src="https://cdn.jsdelivr.net/npm/vue@2.5.17/dist/vue.js"></script>
```

## 在webpack下使用NPM引入
```
$ npm install vue
```

## 使用VUE-CLI引入
```
$ npm install -g @vue/cli
$ vue create projectName
```

## 查看Vue版本
```
$ vue --version
```
